# 📊 Proyecto de Análisis de Datos E-commerce

## 🚀 Descripción general
Este proyecto tiene como objetivo analizar datos transaccionales de un negocio de comercio electrónico para extraer información valiosa que apoye la toma de decisiones.

El análisis incluye limpieza de datos, análisis exploratorio (EDA) y generación de insights de negocio utilizando Python.

---

## 🎯 Objetivos
- Comprender el comportamiento de las ventas en el tiempo
- Identificar los productos y categorías más rentables
- Analizar patrones de compra de los clientes
- Detectar tendencias geográficas en las ventas
- Proponer recomendaciones accionables para el negocio

---

## 🛠️ Tecnologías utilizadas
- Python 🐍
- Pandas
- NumPy
- Matplotlib / Seaborn
- Jupyter Notebook

## 📂 Estructura del proyecto

---

## 🧹 Limpieza de datos
Se realizaron los siguientes procesos:
- Eliminación de valores nulos (CustomerID)
- Eliminación de valores negativos (devoluciones)
- Eliminación de duplicados
- Conversión de formatos de fecha
- Creación de nueva variable: `TotalPrice`

---

## 📊 Análisis exploratorio (EDA)
El análisis busca responder preguntas como:
- ¿Cómo se comportan las ventas mensuales?
- ¿Qué productos generan más ingresos?
- ¿Quiénes son los clientes más importantes?
- ¿Qué países generan más ventas?
